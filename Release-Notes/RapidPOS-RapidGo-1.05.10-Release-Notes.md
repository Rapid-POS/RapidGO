# RapidGo v1.05.10 Release Notes

_Release Date: December 3, 2025_  

---

## New Functionality

### Inventory Adjustment Enhancement
Added a user-facing reminder message to notify when there are unposted inventory adjustments for items.  
- Prompts the user to post the adjustment in Counterpoint after completing the adjustment in RapidGo.  
- Helps prevent discrepancies between RapidGo adjustments and Counterpoint posting.
