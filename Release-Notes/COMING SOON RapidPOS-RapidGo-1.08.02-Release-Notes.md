# RapidGo v1.08.02 Release Notes
**Release Date:** March 24, 2026

## New Functionality

### Gridded Item Functionality Enhancements
Several improvements have been made to gridded item workflows to enhance visibility, accuracy, and usability.

#### Cell Description Display
Gridded items now display the associated cell description.

- Improves visibility and context when working with gridded inventory items.

#### Edit/Delete Scanned Items
Editing and deleting the currently scanned item is now functioning correctly.

- Allows users to reliably manage scanned entries during processing.

#### Grouped Edit List Counts
Editing counts within edit lists are now grouped by scanned datetime.

- Provides better organization and tracking of scanned entries.
- Makes it easier to review and manage count history.

#### Counted Items in Gridded Window
The gridded window now displays counted items.

- Enhances accuracy and transparency during inventory processes.

## Bug Fixes and Performance Enhancements

### Release Template Settings Retrieval Fix
Fixed an issue where app settings were retrieved incorrectly when servers shared the same agent machine name.

- Ensures correct configuration settings are applied per server environment.
