# RapidGo v1.08.01 Release Notes

Release Date: March 10, 2026

## New Functionality

### Toggle Line-Level Comments During Receiving

Users can now toggle **Comment 1**, **Comment 2**, and **Comment 3** at the **line level** when performing a receiving in RapidGo.

* Provides greater flexibility when entering or reviewing comment fields during the receiving process.
* Allows users to enable or disable comment fields as needed for individual line items.

## Bug Fixes and Performance Enhancements

### Sign-In Error Resolved

* Fixed an issue where users could receive the error **“Object reference not set to an instance of an object”** when signing into RapidGo.
* The sign-in process now handles this condition properly, preventing the error message from appearing.
